# 📞 Call Recorder - Twilio + Deepgram

A complete call recording and transcription system using Twilio and Deepgram. Record calls and get automatic transcriptions with speaker identification.

## 🚀 Features

- **Automatic Call Recording** - Record both incoming and outgoing calls
- **Real-time Transcription** - Get transcripts using Deepgram's AI
- **Web Dashboard** - Make calls and view transcripts
- **Mobile Ready** - Backend API ready for mobile apps
- **Cloud Deployed** - Live 24/7 on Render
- **Dual-channel Recording** - Separate agent and customer audio
- **Speaker Diarization** - Identify who said what

## 📋 Prerequisites

- [Twilio Account](https://twilio.com) (Free trial available)
- [Deepgram Account](https://deepgram.com) (Free $200 credit)
- [Render Account](https://render.com) (Free tier available)
- [GitHub Account](https://github.com)

## 🛠️ Tech Stack

- **Backend**: Flask (Python)
- **Telephony**: Twilio API
- **Transcription**: Deepgram AI
- **Deployment**: Render
- **Version Control**: Git/GitHub
